Ecommerce Website

A web-based Ecommerce Website built using Java Servlet, JSP and MySQL.

Tech Stack
- Java
- Java Servlet and JSP
- JDBC
- MySQL
- HTML, CSS and Bootstrap

Features
- User Login and Signup
- Product Display with Categories
- Add to Cart and Remove from Cart
- Session Management
- Database connected with MySQL

How to Run
1. Clone the repository
2. Import project in Eclipse
3. Setup MySQL database
4. Configure DB connection in DBConnection.java
5. Deploy on Apache Tomcat
6. Open browser and go to http://localhost:8080/Ecommerce
